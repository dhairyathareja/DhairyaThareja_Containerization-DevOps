# Assignments – Containerization & DevOps

This section contains all the assignments completed as part of the Containerization and DevOps coursework.
Each assignment is designed to evaluate conceptual understanding, problem-solving ability, and the practical application of DevOps tools and techniques.

The assignments are organized sequentially and focus on strengthening knowledge in containerization, automation, CI/CD pipelines, and deployment strategies through independent work.
